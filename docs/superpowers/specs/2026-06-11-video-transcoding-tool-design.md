# 视频格式转码工具 - 设计文档

> 日期：2026-06-11 | 状态：已确认

## 概述

Windows 桌面视频转码工具，将 MKV 视频无损转换为 MP4 格式，同时支持字幕提取与嵌入。

## 技术栈

| 项目 | 选择 |
|------|------|
| 语言 | Python 3.11+ |
| GUI 框架 | PyQt6 |
| 转码引擎 | FFmpeg（内置打包） |
| 打包工具 | PyInstaller |

## 架构

```
GUI 层 (gui/)          ← PyQt6 界面，主窗口 + 任务列表 + 日志 + 设置
    ↓
任务调度层 (core/scheduler.py)  ← 批量任务队列、并发控制、状态追踪
    ↓
FFmpeg 引擎层 (core/engine.py)  ← FFmpeg 命令构建、子进程调用、进度解析
    ↓
FFmpeg 二进制 (ffmpeg/ffmpeg.exe)
```

## 核心功能

### 1. MKV → MP4 转码（流复制）

```
ffmpeg -i 输入.mkv -map 0:v -c:v copy -map 0:a -c:a copy 输出.mp4
```

- 视频/音频流复制，不重新编码，无损无压缩
- 瓶颈为磁盘 I/O，速度极快

### 2. 字幕提取

```
ffmpeg -i 输入.mkv -map 0:s:N 输出.srt
```

- 从 MKV 中提取指定字幕轨道
- 统一输出为 SRT 格式

### 3. 字幕嵌入 MP4

```
ffmpeg -i 输入.mkv -i 字幕.srt -map 0:v -c:v copy -map 0:a -c:a copy -map 1:s -c:s mov_text 输出.mp4
```

- 使用 MP4 兼容的 mov_text 编码
- 可选设置语言元数据

## 用户可选选项

| 选项 | 默认 |
|------|------|
| 导出独立 SRT 字幕文件 | ✅ 勾选 |
| 嵌入字幕到 MP4 | ✅ 勾选 |

两个选项独立，可自由组合。

## 多轨字幕处理

- 默认处理所有字幕轨道
- 可切换为手动选择需要处理的轨道

## 任务调度

- FIFO 队列，默认单任务串行
- 状态：等待中 / 进行中 / 已完成 / 失败
- 失败不阻塞后续任务
- 进度通过解析 FFmpeg stderr 的 `time=` 字段计算

## 打包发布

- PyInstaller 打包为单个 .exe
- FFmpeg 二进制作为外部资源包含在安装目录
